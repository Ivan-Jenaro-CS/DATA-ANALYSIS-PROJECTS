I conducted a rainfall analysis based on a dataset from an environmental station to generate predicted storms using the Monte Carlo method. 

According to this dataset, it was observed that during the March–October period there is significant rainfall, and historically there have been days with the highest precipitation levels between 1959 and 2019.

The WMO defines rainfall above 50 mm/hour as heavy rainfall. Figure 2 shows the frequency of days with heavy rainfall across defined ranges on the X-axis.

This statistical analysis helped determine the rainfall intensity used in the creation of the storms.
